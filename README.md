# MobileAppDev — Mobile app examples

This repository contains small mobile app projects used for learning and demos.

Top-level folders
- `Swift` — SwiftUI iOS apps (Xcode project). See `Swift/MyFavouritePlaces` for My Favourite Places app.
- `ReactNative` — React Native / Expo projects. See `ReactNative/pocket-lists` for the Pocket Lists app.

Quick start

1. Open the iOS app in Xcode:

   - Open `Swift/MyFavouritePlaces/MyFavouritePlaces.xcodeproj` in Xcode.
   - Select a simulator or device and press Run.

2. Run the React Native / Expo app (`pocket-lists`):

   ```bash
   cd ReactNative/pocket-lists
   npm install
   npx expo start
   ```
